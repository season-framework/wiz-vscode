# 032 - 프로젝트 삭제 기능 추가

## 개요
로컬 프로젝트 폴더 삭제 기능 구현

## 요구사항
불필요한 프로젝트를 VS Code 내에서 삭제하는 기능

## 구현

`src/extension.js`:

### 1. 메뉴 항목 추가
"프로젝트 불러오기" 아래에 "프로젝트 삭제하기" 항목 추가

### 2. 안전 장치
- 삭제할 프로젝트 선택 단계 분리
- `showWarningMessage`로 최종 확인 절차 진행

### 3. 실행
- `fs.rmSync`로 프로젝트 폴더 영구 삭제
- 현재 활성 프로젝트 삭제 시 다른 프로젝트로 전환 로직 포함
