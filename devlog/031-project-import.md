# 031 - 프로젝트 불러오기 기능 추가

## 개요
Git 저장소 URL을 통해 새 프로젝트 복제/불러오기

## 요구사항
탐색기 상단 메뉴에서 Git 저장소를 복제하여 새 프로젝트로 불러오는 기능

## 구현

`src/extension.js`:

### 1. 메뉴 항목 추가
프로젝트 목록 퀵픽(QuickPick) 상단에 "프로젝트 불러오기" 항목 추가

### 2. 입력 프로세스
- 프로젝트 이름 입력 (영문 소문자+숫자, 중복 체크)
- Git Repository URL 입력

### 3. 실행
- `child_process.exec`로 `git clone` 명령 실행
- `project/<입력한이름>` 경로에 저장소 복제
- 완료 후 즉시 전환 여부 확인
