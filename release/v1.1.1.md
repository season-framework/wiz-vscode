# Wiz VSCode Extension v1.1.1 Release Notes

## Summary

**v1.1.1** delivers bug fixes and usability improvements for the build system, keyboard navigation, and URI compatibility.

---

## Bug Fixes

### Auto-Build Condition Fix
- Auto-build now only triggers when file content has actually changed (not on every save)
- Fixed `wiz://` tab activation triggering unnecessary builds

### Source Angular Tree ID Duplicate Fix
- Resolved tree item ID collision in Source Angular apps that caused expand/collapse errors

### Wiz URI Compatibility
- Improved `wiz://` URI handling for compatibility with external extensions and editor features

---

## Improvements

### Python Interpreter Selection
- Added Python interpreter selection for venv/conda build environments
- Wiz executable path resolution improved with fallback logic

### Keyboard Navigation Overhaul
- Replaced Alt+1–6 shortcuts with Opt+A / Opt+S / Opt+T (Mac) and Alt+A / Alt+S / Alt+T (Win/Linux)
- **Opt+A / Alt+A**: Navigate to previous file type
- **Opt+S / Alt+S**: Navigate to next file type
- **Opt+T / Alt+T**: Open current document in split view
- File type cycling: UI → Component → SCSS → API → Socket
